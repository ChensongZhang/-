* ABC

1. 呵呵
2. 哈哈
3. 嘿嘿
4. 哼哼
 
 
&lt;   // 会显示为”<“
&amp;  // 会显示为”&“：在 href 属性里面，必须将 & 转变为 &amp;
\.     // 为了防止产生"1."变为有序列表，则可以写成"1\."
 * _   // 如果 * 和 _ 两边都有空白的话，它们就只会被当成普通的符号。
